# Steam Like Database

Steam Like Database is a learning project that simulates a simplified version of the Steam platform's backend.  
It allows users to practice SQL queries on a dataset of users and games.

## Features
- Users and games tables with sample data
- Execute SQL queries to retrieve, filter, and analyze data
- Practice SELECT, JOIN, INSERT, UPDATE, DELETE operations

## How to Run
1. Clone this repository:
git clone https://github.com/ycnkc/steamlikedatabase.git
2. Open the SQL files in your preferred database tool (e.g., MySQL Workbench, SQLite)
3. Execute queries on the sample datasets

## Technologies
- SQL
- SQLite / MySQL
- Sample datasets included

## Notes
- This project is intended as a learning tool to practice database queries and understand relational database concepts.
